Advanced Stock Summary Generator

How to use:
1. Open stock-summary-generator.html in any browser.
2. Drag and drop your WINGS STOCK Excel file.
3. Generate reports automatically.
4. Download all summaries as Excel ZIP or PDF ZIP.

Features:
- All models and all years.
- Filters by year and branch.
- Custom report title.
- Print-ready yellow header design.
- Export each summary to separate Excel files.
- Export each summary to separate PDF files.
- One-click ZIP downloads.
- Works fully offline after opening the HTML file.
